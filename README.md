## 怎么用 Desktop Composer：从零开始配置第一个主题

整个流程非常直接，分两步。

**第一步：创建配置文件**

1. 下载安装 Desktop Composer，启动后会看到一个主界面，左侧是配置文件列表
2. 点击左侧「+」号，新建一个配置文件，给它起个名字（比如"深夜编程"）
3. 在右侧的配置区域，逐项选择你想保存的设置——壁纸、明暗模式、Dock 状态、要同步切换的 VS Code 主题，等等
4. 如果某项设置你不想让它被切换（比如你永远用固定壁纸），可以在该项旁边关掉同步开关
5. 保存

**第二步：切换配置文件**

- 在主界面点击任意一个配置文件右上角的启动按钮，所有设置立刻同步生效
- 或者点击菜单栏的 Desktop Composer 图标，在下拉列表里一键切换
- 还可以在「设置」里配置定时规则，比如每天 9:00 自动切换到"工作模式"，22:00 自动切换到"护眼模式"

Setapp 平台上有用户评价这款软件的使用体验是"当我在设置里翻到它支持同步 VS Code 主题时，简直被震惊到了"（来源：Setapp 用户评价，Christopher Cook）。
